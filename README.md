ローグライクグリットバトルカードゲーム

グリッド型戦術ゲームのための C#ロジック中心アーキテクチャを試作するプロジェクトです。


本プロジェクトでは ゲームロジックを純C#で実装し、Unityは描画・演出レイヤーとして利用する設計を採用しています。

Unityプロジェクトでありがちな MonoBehaviour依存のロジック肥大化を避け、

拡張性・テスト容易性を重視した設計を目指しています。

なお、小生は業務系アプリエンジニアであるためUnityについては現在学習段階です。至らない点があると存じますがご了承ください。

---

特徴

このゲームの特徴：

・Slay the Spireを参考にした、ローグライクカードバトルです。

・プレイヤーは５*5のフィールドで、毎ターンカードを使用して戦闘を行います。

・移動、攻撃、その他アクションはすべてカードを使用します。
