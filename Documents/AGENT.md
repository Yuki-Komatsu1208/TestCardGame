# AGENT.md

この文書は、本リポジトリで作業するAIエージェント向けの実装ルールを定義する。

## 基本方針

- Unity AI は、特別な指示がない限り Unity Asset の生成のみを行う。
- Codex およびその他のエディタ統合型AIは、開発者の明示的な指示に基づいて実装を行う。
- 全体設計、アーキテクチャ設計、ゲームデザイン、仕様判断は、すべて開発者が行う。

## 役割分担

- AIは独自判断で設計を変更しない。
- AIは独自判断でゲームルールやバランスを決めない。
- AIは既存設計に従って、実装、補助、整理、記述を行う。
- 設計上の不足や矛盾を見つけた場合は、勝手に補完せず開発者に確認する。

## 実装ルール

- 実装は必ず開発者の指示を起点とする。
- 指示されていない機能追加、仕様変更、リファクタリングは行わない。
- 既存コードの意図を尊重し、必要最小限の変更に留める。
- アセット生成を担当するAIは、コード実装や設計判断を行わない。

## 記述ルール

- コメントは日本語で記述する。
- エラーメッセージは日本語で記述する。
- 実装時の補助説明やログも、可能な限り日本語で統一する。

## 禁止事項

- AIが設計者として振る舞うことを禁止する。
- AIが開発者の指示なしに仕様を確定することを禁止する。
- AIが独断で責務分離や構造変更を進めることを禁止する。
