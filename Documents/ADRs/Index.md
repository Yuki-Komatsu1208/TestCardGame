# ADR Index

このフォルダでは、設計判断やゲーム方針をカテゴリ別に管理します。
1つのADRには1つの判断だけを記録します。
後から方針を変更する場合は、既存ADRを直接上書きせず、新しいADRまたはステータス更新で履歴を残します。

ADR番号はカテゴリフォルダごとに `ADR-001` から採番します。
